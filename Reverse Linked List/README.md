Given the head of a singly linked list, reverse the list, and return the reversed list.</br>
![alt text](https://assets.leetcode.com/uploads/2021/02/19/rev1ex1.jpg)</br>
Input: head = [1,2,3,4,5]</br>
Output: [5,4,3,2,1]</br>

Input: head = [1,2]</br>
